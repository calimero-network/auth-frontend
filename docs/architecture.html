Architecture

React 19 flows — AdminFlow, ApplicationFlow, PackageFlow — backed by mero-js v2

Source Structure

src/App.tsx
Root. Reads the app mode from URL params to determine which flow to run (admin vs application vs package). Checks for SSO tokens in URL hash first.
src/flows/AdminFlow.tsx
Flow for node admin users. Direct node credential login with admin-level access.
src/flows/ApplicationFlow.tsx
Flow for app users. Login → context selection → permission grant → redirect back to app.
src/flows/PackageFlow.tsx
Flow for package/registry operations requiring elevated auth.
src/components/auth/LoginView.tsx
Login form component. Accepts node URL + credentials. Calls mero-js auth API. Key file for SSO: checks URL hash for tokens before showing form.
src/components/
Shared components: ContextSelector, PermissionsView, StatusBadge, loading states.
src/hooks/
Custom hooks: useAuth, useContexts, usePermissions.

Flow State Machines

Each flow is a state machine. ApplicationFlow steps:

check_hash
SSO tokens?
login
credential entry
context
select / create
done
redirect to app

If SSO tokens are present in the URL hash, the flow jumps directly to done — skipping login and context selection.

mero-js v2 Integration

The auth frontend uses @calimero-network/mero-js v2 (breaking API change from the older calimero-client). Key v2 differences:

GenerateClientKeyRequest
Uses camelCase fields: contextId, contextIdentity (v1 used context_id, context_identity).
Auth flow
Login returns { access_token, refresh_token } directly. Token storage via TokenStore interface.
Admin API
Context and identity management via sdk.admin.* methods instead of separate class.

Tech Stack

React 19

Functional components, hooks, Suspense boundaries for loading states.

TypeScript

Strict mode. All API types imported from mero-js.

Vite

Fast HMR, optimised build output. Served as a static bundle by the node's web server.

Emotion

CSS-in-JS for component styles. Theme tokens for consistent design.

React Router DOM

Navigation between steps in multi-step flows.

pnpm

Package management. Lock file committed.
