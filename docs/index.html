What Auth Frontend Does

Auth Frontend is the UI layer that apps load when they need to authenticate a user with a Calimero node. It is served as a standalone built app and embedded via the node's app registry.

1

Node Login

User enters node URL and credentials. The frontend exchanges them for JWT tokens via mero-js and stores them securely.

2

Context Selection

User selects an existing context or creates a new one. Each context is an isolated execution environment on the node.

3

Permission Grant

User approves the requesting app's required permissions before the auth flow completes and redirects back to the app.
